¿Qué es el LI-FI?

LI-FI (Light Fidelity) es una tecnología innovadora que utiliza la luz visible para transmitir datos de manera inalámbrica. A diferencia del Wi-Fi, que emplea ondas de radio, LI-FI aprovecha las variaciones en la intensidad de la luz LED, invisibles para el ojo humano, para codificar y enviar información a dispositivos receptores. Esta tecnología promete velocidades más rápidas, mayor seguridad y una menor congestión de las frecuencias tradicionales.

Objetivo del Proyecto

El objetivo de este proyecto es construir un sistema básico de comunicación LI-FI, donde se logre la transmisión de datos simples, como texto, a través de la luz. Se busca demostrar cómo la luz LED puede ser modulada para transferir información y ser interpretada por un receptor fotosensible.

Pasos del Proyecto

  1. Investigación: Comprender cómo funciona la transmisión de datos por luz y sus ventajas.
  2. Materiales: LED, fotodiodo, Arduino (o microcontrolador), cables y protoboard.
  3. Montaje: Conectar el LED al transmisor y el fotodiodo al receptor.
  4. Programación: Escribir el código para modular la luz y decodificar los datos en el receptor.
  5. Pruebas: Enviar texto o datos simples y verificar la transmisión a través de la luz.
